Arrays

An array is Array structure which can hold multiple elements of one data type each located at indices. Arrays are 0-indexed, meaning the first index is index 0, the second is 1, and so on. The last index of an array will be its length minus one. Arrays have a fixed size, but their contents are not fixed. The array data type is specified by brackets '[]' following whatever data type it is supposed to hold. To instantiate an array, a size must be set for it. 

 int[] numbers = new int[10];
 String[] words = new String[999];

ArrayLists

An ArrayList is a collection which is similar to an array in that it has elements with indices. But, it can only hold objects (unlike the array which can hold primitive datatypes as well). ArrayLists also don't have a fixed size, elements can be dynamically added or removed. Whenever an element must be added or removed, the size is automatically adjusted by the ArrayList object. The object type the ArrayList holds is enclosed by chevrons '<>'.

The for loop is used to navigate arrays. Array objects have a length variable which can be accessed publicly via the dot operator (like in any object). So any array's elements can be printed using the following loop: 

 for (int i = 0; i < array.length; i++) {
     System.out.println(array[i]);
 }

A for-each loop (enhanced for loop) is a type of loop that retrieves a collection's elements. For each iteration of the loop, the next element is stored in the loop variable. It is a more convenient way of accessing an array's elements than the example shown above. It can be used with the following syntax: 

 for (int i : array) {
     System.out.println(i);
 }

This works as long as the array is an integer array. If the array has elements of a different datatype, the loop variable's data type must be changed accordingly. The for each loop can be used for any collection, which includes both arrays and ArrayLists.

Below is my example from a class assignment of a method that prints the point values of some Cards. 

  /**
   * Calculates points for hands and outputs sorted results.
   */
  private static void evaluate() {
    for (int i = 0; i < 4; i++) {
      if (!bad[i]) { // if hand is valid
        int points = 0;
        ArrayList spades = new ArrayList();
        ArrayList hearts = new ArrayList();
        ArrayList clubs = new ArrayList();
        ArrayList diamonds = new ArrayList();

        for (int j = 0; j < 13; j++) {
          Card card = new Card(cards[i][j].rank, cards[i][j].suit);
          char temp = cards[i][j].rank;
          if (temp == 'J') {
            points++;
          } else if (temp == 'Q') {
            points += 2;
          } else if (temp == 'K') {
            points += 3;
          } else if (temp == 'A') {
            points += 4;
          }
          temp = cards[i][j].suit;
          if (temp == 'S') {
            spades.add(card);
          } else if (temp == 'H') {
            hearts.add(card);
          } else if (temp == 'D') {
            diamonds.add(card);
          } else if (temp == 'C') {
            clubs.add(card);
          }
        }

        // add points for number of suits
        points += suitNum(spades);
        points += suitNum(hearts);
        points += suitNum(diamonds);
        points += suitNum(clubs);

        // output
        System.out.println("\nHand " + (i + 1));

        System.out.print("Spades: ");
        if (spades.size() > 1) {
          sort(spades);
        }
        for (Card c : spades) {
          System.out.print(c + " ");
        }

        System.out.print("\nHearts: ");
        if (hearts.size() > 1) {
          sort(hearts);
        }
        for (Card c : hearts) {
          System.out.print(c + " ");
        }

        System.out.print("\nDiamonds: ");
        if (diamonds.size() > 1) {
          sort(diamonds);
        }
        for (Card c : diamonds) {
          System.out.print(c + " ");
        }

        System.out.print("\nClubs: ");
        if (clubs.size() > 1) {
          sort(clubs);
        }
        for (Card c : clubs) {
          System.out.print(c + " ");
        }

        System.out.println("\nTotal points: " + points);
      }
    }
  }
